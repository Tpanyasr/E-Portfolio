  • Recipe Buddy

    Html, CSS, JavaScript, React, Node, MongoDB

    RecipeBuddy is a full-stack recipe book where everyone can connect and view each other's recipes. With this, the user can see the total cost of the meal and price per serving of their recipe. It is great for people who want to budget and be inspired to cook at home rather than eat out!

  • Meme-ED

    Html, CSS, Atom, Bootstrap

    Meme-Ed is a student built website which focuses on helping students clarify and assure their existing knowledge, as well as learning new information, through memes. This project was submitted to Venus Hacks and was a great intro project to HTML, CSS, and web dev!

  • Pokemon Game

    Java, JUnit, AStar Pathing

    This pokemon game was created in my object-oriented-programming class with the goal of learning OOP concept such as refactoring, abstraction, and pathing algorithms such as Dijkstra's Algorithm and AStar Pathing. The goal of the game is to run around catching pokemon, and being able to evolve them using specific key presses. In addition, the game includes a world changing mouse trigger!

  • Dream Higher

    Figma, Design Thinking, UX Research/Design, Coda

    I collaborated with a cohort of 5 UX designers and 2 project managers to create Dream Higher! The purpose of Dream Higher is to provide educational opportunities for undocumented students through personalized support. Throughout this case study we started with a problem statement, conducted research, prototyped, and iterated through multiple versions to create a high fidelity prototype.

  • Hack4Impact: Lumina Alliance Team

    Typescript, AWS Amplify, React, Express, Chakra UI

    Hack4Impact is an organization that works with non-profit organizations to build powerful, impactful software as a tool for social good. My team is currently working with Lumina Alliance which is an organization whose purpose is to empower those impacted by sexual and intimate partner violence through innovative advocacy, healing, and prevention programs. Our team of software developers and a PM are currently working to create a scheduling desktop/mobile application for keeping track of volunteers.
